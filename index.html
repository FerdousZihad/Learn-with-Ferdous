Flexbox and Grid both are the Css layout. We can use use these layout for formatting elements inside the layout .

Flexbox is a one dimensional layout . It works only for a row .Using flex we can set the directions row wise or column wise. But the CSS Grid is a two dimensional layout . It is much more flexible then Flexbox and can works in multi-dimensional layout.
That is the main difference between flexbox and grid.

Bootstrap and Tailwind are both CSS Framework comes with thousands of Class to design a responsive website without using any extra CSS.

Bootstrap is a 12 grid system based CSS framework which focus on responsiveness . But Tailwind is utility based CSS framework which focus on utilities. Tailwind is much more flexible to design,Bootstrap much more flexible to make Website responsive.

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
The total width of an element should be calculated like this:
Total element width = width + left padding + right padding + left border + right border + left margin + right margin
The total height of an element should be calculated like this:
Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
